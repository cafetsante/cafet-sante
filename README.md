# Cafet Santé

Dépôt statique pour héberger le logo et les icônes de l’application.

## Publication avec GitHub Pages

1. Créer un dépôt public nommé `cafet-sante` sur le compte `cafetsante`.
2. Importer tous les fichiers de ce dossier à la racine du dépôt.
3. Ouvrir **Settings → Pages**.
4. Dans **Build and deployment**, choisir **Deploy from a branch**.
5. Sélectionner la branche `main` et le dossier `/ (root)`.
6. Enregistrer.

Après publication, les ressources seront disponibles ici :

- Page : https://cafetsante.github.io/cafet-sante/
- Logo 512 px : https://cafetsante.github.io/cafet-sante/icon-512.png
- Logo 192 px : https://cafetsante.github.io/cafet-sante/icon-192.png
- Icône Apple : https://cafetsante.github.io/cafet-sante/apple-touch-icon.png
- Favicon : https://cafetsante.github.io/cafet-sante/favicon-32x32.png
- Manifest : https://cafetsante.github.io/cafet-sante/manifest.webmanifest
